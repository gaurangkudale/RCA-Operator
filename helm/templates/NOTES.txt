RCA Operator has been installed successfully!

1. Verify the operator is running:
   kubectl get pods -n {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "rca-operator.name" . }}"

2. Create the OpenAI API secret (if using AI-powered RCA):
   kubectl create secret generic rca-agent-openai-secret \
     --from-literal=apiKey=<YOUR_OPENAI_API_KEY> \
     -n {{ .Release.Namespace }}

3. Create your first RCAAgent:
   cat <<EOF | kubectl apply -f -
   apiVersion: rca.rca-operator.tech/v1alpha1
   kind: RCAAgent
   metadata:
     name: my-rca-agent
     namespace: {{ .Release.Namespace }}
   spec:
     watchNamespaces:
       - default
     retention: "168h"  # 7 days
   EOF

4. Check for incident reports:
   kubectl get incidentreports -A

For more information, visit:
  https://github.com/gaurangkudale/RCA-Operator
